# Level Width

## ❔
기존에 구현한 클래스 `Node`를 바탕으로 주어진 트리에서 각 레벨의 노드 수를 구하는 `levelWidth` 함수 구현
- 매개변수: 배열
- 반환값: 배열

## ✔️
배열의 요소로 `"stop"`을 설정한 후 이동하여 각 레벨의 노드 수 측정  
배열의 요소로 `"stop"`만 남을 경우 반복문 종료
